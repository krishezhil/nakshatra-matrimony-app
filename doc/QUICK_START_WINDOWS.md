### What You Received

### Step 1: Launch the Application
1. **Double-click** `matrimony-app.exe`
2. **Wait** for the application to start (may take 30-60 seconds on first launch)
3. **Allow** Windows Defender/Firewall access if prompted
4. **Open your browser** to `http://localhost:3131` (should open automatically)

### Step 2: First Time Setup
1. **Create your first profile** using the "Create Profile" button
2. **Fill in all required fields** (Name, Age, Nakshatra, etc.)
3. **Save the profile** - you'll see confirmation message

### Step 3: Key Features
- **Search Profiles**: Find profiles by various criteria
- **Find Matching**: Advanced nakshatra-based compatibility matching
- **Export Options**: Generate PDF or Excel reports
- **WhatsApp Sharing**: Share profiles directly via WhatsApp

### Step 4: Data Location
Your profile data is safely stored at:
`C:\Users\[YourUsername]\AppData\Roaming\matrimony\data\`

## üîß Troubleshooting

### Application Won't Start
1. **Check if port 3131 is free** - close other applications using this port
2. **Run as Administrator** - right-click exe file ‚Üí "Run as administrator"
3. **Check Windows Defender** - ensure the file isn't blocked

### Browser Doesn't Open Automatically
1. **Manually open browser** and go to `http://localhost:3131`
2. **Check Windows Firewall** - allow the application through firewall
3. **Try different browser** - Chrome, Firefox, or Edge

### Cannot Create Profiles
1. **Check folder permissions** - ensure app can write to AppData folder
2. **Run as Administrator** if permission issues persist
3. **Restart the application** and try again
