## Getting Started

### System Requirements
- **Operating System**: Windows 10/11
- **Browser**: Chrome (recommended), Firefox, Edge
- **Storage**: Minimum 100MB free space
- **Memory**: 2GB RAM minimum

### First Launch
1. **Start the Application**:
   - Double-click `nakshatra-matrimony-win64.exe` (if using executable)
   - Or run `npm start` (if using development version)
   
2. **Automatic Browser Opening**:
   - Chrome will automatically open in app mode
   - Application loads at `http://localhost:3131`
   - If browser doesn't open, manually navigate to the URL

3. **Welcome Screen**:
   - Click **"Enter Site"** to access the main application
   - You'll see the main dashboard with navigation options

## Creating Profiles

### Profile Creation Process

1. **Navigate to Create Profile**:
   - Click **"Create Profile"** from main menu
   - Or use the navigation bar link

2. **Personal Information Section**:
   - **Name**: Full name (required)
   - **Gender**: Male/Female selection (required)
   - **Contact**: Phone number with validation (required)
   - **Email**: Email address (required)

3. **Birth Details Section**:
   - **Birth Date**: Date picker (required)
   - **Birth Time**: Time selection (required, now mandatory)
   - **Birth Place**: Location of birth (required, now mandatory)
   - **Age**: Automatically calculated from birth date

4. **Astrological Information**:
   - **Nakshatra**: Select from dropdown (required)
   - **Rasi/Lagnam**: Zodiac sign (required)  
   - **Gothram**: Family lineage (required)

5. **Family & Background**:
   - **Father's Name**: Parent information (required)
   - **Family Details**: Additional family information
   - **Occupation**: Professional details (required)
   - **Income**: Salary/income information (required)

6. **Location & Preferences**:
   - **Current Location**: Where they live (required)
   - **Education**: Educational qualifications (required)
   - **Marital Status**: Single, Divorced, Widowed (required)

### Validation Features
- **Real-time Validation**: Immediate feedback on form fields
- **Required Field Indicators**: Clear marking of mandatory fields
- **Format Checking**: Phone numbers, emails, dates validated
- **Indian Name Support**: Handles names with parentheses and special characters

### Form Completion
- Click **"Create Profile"** to save
- Success confirmation with profile serial number
- Automatic redirection to profile list

## Data Backup & Restore

### Backup System

Protect your valuable matrimonial data with comprehensive backup features:

#### **Creating Backups**:

1. **Navigate to Data Backup**:
   - Click **"Data Backup"** from main menu
   - Access backup administration panel

2. **Backup Options**:
   - **Full Backup**: Complete profile database
   - **Incremental**: Changes since last backup
   - **Selective**: Choose specific data types

3. **Backup Process**:
   - Click **"Create Backup"** 
   - System creates timestamped backup file
   - Backup saved to secure location

#### **Backup Contents**:
- All profile data (personal, family, astrological)
- Nakshatra and gothram reference data
- System configuration settings
- Application preferences

#### **Backup Storage**:
- **Location**: `%APPDATA%\matrimony\backups\`
- **Format**: Compressed JSON files
- **Naming**: Timestamp-based file names
- **Security**: Local storage for privacy

### Restore Operations

#### **Restoring from Backup**:

1. **Select Restore Option**:
   - Access backup administration panel
   - Click **"Restore from Backup"**
   - Browse available backup files

2. **Restoration Process**:
   - Choose backup file to restore from
   - System validates backup integrity
   - Confirms restoration completion

3. **Data Recovery**:
   - All profiles restored to backup state
   - System references updated
   - Application ready for immediate use

#### **Backup Best Practices**:
- **Regular Backups**: Weekly backup schedule recommended
- **Version Control**: Keep multiple backup versions
- **Secure Storage**: Store backups in safe location
- **Test Restores**: Verify backup integrity periodically

## Troubleshooting

### Common Issues and Solutions

#### **Application Startup Issues**:

**Problem**: Application won't start
- **Solution 1**: Check if port 3131 is already in use
- **Solution 2**: Run as administrator if permission issues
- **Solution 3**: Verify all required files are present

**Problem**: Browser doesn't open automatically
- **Solution**: Manually navigate to `http://localhost:3131`
- **Alternative**: Check Chrome installation and permissions

#### **Data and Profile Issues**:

**Problem**: Profiles not displaying
- **Solution 1**: Check data file permissions
- **Solution 2**: Verify profile data format integrity
- **Solution 3**: Restore from recent backup if needed

**Problem**: Search results empty
- **Solution 1**: Verify search criteria settings
- **Solution 2**: Check profile database contains data
- **Solution 3**: Clear search filters and try again

#### **Export and Backup Issues**:

**Problem**: Export files not generating
- **Solution 1**: Check Downloads folder permissions
- **Solution 2**: Verify sufficient disk space available
- **Solution 3**: Try different export format

**Problem**: Backup restore failing
- **Solution 1**: Verify backup file integrity
- **Solution 2**: Check backup file format compatibility
- **Solution 3**: Contact support if persistent issues

#### **Performance Issues**:

**Problem**: Application running slowly
- **Solution 1**: Close unnecessary browser tabs
- **Solution 2**: Restart application periodically
- **Solution 3**: Ensure adequate system resources

**Problem**: Large database handling
- **Solution 1**: Use filtering options to reduce result sets
- **Solution 2**: Regular backup and cleanup procedures
- **Solution 3**: Consider data archiving for old profiles

### Getting Additional Help

#### **Data Safety**:
